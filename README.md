A JSON with color names and its values. Based on http://dev.w3.org/csswg/css-color/#named-colors.

[![NPM](https://nodei.co/npm/color-name.png?mini=true)](https://npmjs.org/color-name/)


```js
import colors from 'color-name';
colors.red // [255,0,0]
```
